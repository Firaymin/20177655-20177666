使用说明：
点击github上的“Clone or download”按钮后，将上述文件打包下载，解压后即可使用，需保证上述所有文件在同一个文件夹下。
使用时直接用Chrome打开学术家族树.html文件即可，输入格式按照作业要求即可。
在出现的文本框输入数据，点击生成家族树按钮，将会在下方生成一棵以导师为根节点的树（暂不支持关联树的实现...)，然后点击刷新按钮可以刷新页面重新输入。
